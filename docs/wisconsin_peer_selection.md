# Wisconsin Peer Selection

## Default Peer Set

The advanced notebook uses this first-pass peer group for Wisconsin:

- `MN`
- `IA`
- `IL`
- `MI`
- `IN`
- `OH`

## Why These States

- Upper Midwest and nearby regional context
- broadly comparable legal and healthcare market context for an initial descriptive benchmark
- enough malpractice volume to support stable decade-level summaries
- practical relevance for Wisconsin-focused interpretation

## Caveat

This is a defensible starting comparison set, not a final causal-control design. If the team later adds external covariates such as population, physician density, or tort-reform variables, peer selection can be refined.

## Reporting Rule

Wisconsin outputs should always be shown in two ways:

- Wisconsin alone over time
- Wisconsin against the peer group
